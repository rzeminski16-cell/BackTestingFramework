# Securities Reference

This document lists the available securities in the framework's default data universe.

---

## Overview

The framework supports backtesting on:
- **165+ US Equities** — Large, mid, and small cap stocks
- **46 ETFs** — Index, sector, and thematic ETFs
- **Forex Pairs** — Major currency pairs

Data is sourced from Alpha Vantage and stored in `raw_data/daily/`.

## Adding New Securities

To add securities not in the default list:

1. Add ticker to `config/data_collection/tickers.json`
2. Run Data Collection GUI to fetch data
3. Data will be saved to `raw_data/daily/{TICKER}_daily.csv`
